# Image Generation

Generate images from structured requirements. Use this for concept art, icons, character outputs, and other visual assets.

## Usage Notes

- Expand user requests into production prompts before calling the tool.
- Keep strict negative constraints to prevent extra objects or clutter.
- For cutout deliverables, force one solid key background color to support background removal.
